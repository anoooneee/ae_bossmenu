A modern, fully open-source Boss Menu system for your FiveM server. Designed for easy management of employees, finances, and faction hierarchy.

---

## ✨ Features
- 💼 **Employee Management** – Recruit, fire, and promote members with ease.
- 💰 **Faction Finances** – Transparent logs for all deposits and withdrawals from the faction fund.
- 📊 **Modern UI** – A clean, intuitive design focused on user experience.
- ⚙️ **Easy Configuration** – Highly customizable settings directly in the config file.
- 🛠️ **Open Source** – Feel free to modify and adapt the script to your server's needs.
